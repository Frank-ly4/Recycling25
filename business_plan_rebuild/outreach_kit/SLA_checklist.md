# SLA & Operations Checklist

## Driver Daily Checklist
- [ ] Vehicle Inspection (Tires, Fluids, Lights).
- [ ] Scale Calibration (Zero check).
- [ ] PPE Check (Vest, Gloves, Boots).
- [ ] Route Plan Loaded in App.

## Site Pickup Protocol
1.  **Arrival:**
    - [ ] Park safely in designated bay.
    - [ ] Greet Security/Juristic.
2.  **Intake:**
    - [ ] Visual check of bins (Photo taken).
    - [ ] **Contamination Check:** Is it <15%? (If No, alert Juristic immediately).
    - [ ] Weigh bags individually.
    - [ ] Record weights.
3.  **Loading:**
    - [ ] Secure load in cage.
    - [ ] Return bins to designated spot.
    - [ ] Sweep area (Leave it cleaner than found).
4.  **Departure:**
    - [ ] Sign operational log with Juristic.

## Monthly Reporting Checklist (Admin)
- [ ] Aggregate weights per building.
- [ ] Calculate CO2e offset (Weight x Factor).
- [ ] Flag contamination incidents.
- [ ] Generate PDF Report.
- [ ] Email to Juristic Manager by 5th of month.
