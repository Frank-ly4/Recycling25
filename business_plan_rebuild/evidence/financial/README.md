# Financial Evidence (Controlled Access)

This folder is for **company-specific** financial artifacts that support diligence. Prefer **redacted** copies here, with unredacted originals stored in a controlled-access vault.

## What belongs here
- Bank statements / proof of funds (redacted)
- Filed tax returns and payment receipts (VAT/WHT/CIT as applicable)
- Accounting exports / actuals (if any) + supporting receipts/invoices
- Accountant letter (optional)

## Suggested naming convention
- `YYYYMMDD_<topic>_<redacted|unredacted>_<notes>.pdf`
  - Example: `20251224_bank_statement_redacted_last3mo.pdf`

## Redaction baseline
- Redact: account numbers, personal IDs, home addresses, signatures (unless required).
- Keep: date ranges, balances (if acceptable), entity names, bank name, official stamps (if present).
